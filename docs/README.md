## Criar modelagem conceitual

1. Pressione o botão `NOVA MODELAGEM`

    <img src="imgs/createmodel1.png"

        title="Criar modelagem">
</div>

2. Preencha o campo `Título` 
3. Selecione a tipo `Conceitual` 
4. Pressione o botão `SALVAR`
